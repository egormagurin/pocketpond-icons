# Pocket Pond — Steam Inventory Icons

Public icon assets for the **Pocket Pond** Steam Inventory items (AppId `4851300`).

Each `items/Fish_<Name>.png` is a 200×200 icon used as the `icon_url` of a Steam Inventory
item definition. Steam downloads and caches these when the item definitions are published, so
this repo only needs to be reachable at publish/update time.

**Icon URL base:**
`https://raw.githubusercontent.com/egormagurin/pocketpond-icons/main/items/`

(or enable GitHub Pages → `https://egormagurin.github.io/pocketpond-icons/items/`)

Generated from the game's `FishData` sprites via **Tools ▸ Fishing ▸ Export Steam Icons**.
This repo is intentionally **public** and kept separate from the private game source.
